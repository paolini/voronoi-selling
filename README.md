## Descrizione

# voronoi-selling
3D visualization app to display the Voronoi diagram of a lattice cell given by Selling parameters.

## Stack Tecnologico

- **Frontend:** React + Vite
- **Linguaggio:** JavaScript/TypeScript
- **Tooling:** ESLint, HMR, plugin React

## Avvio rapido

1. Installa le dipendenze:
   ```bash
   npm install
   ```
2. Avvia il server di sviluppo:
   ```bash
   npm run dev
   ```
